# Well-mind-ai
# Sleep Quality Predictor

Machine learning project to predict sleep quality based on lifestyle and health parameters.

## Key Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model training and evaluation
- Random Forest and XGBoost model comparison
- Model serialization using Pickle

## Tech Stack
Python, Pandas, NumPy, scikit-learn, Random Forest, XGBoost, Matplotlib, Seaborn 
, Streamlit

## Notes
- Model training and evaluation were performed using Google Colab.
- The complete ML pipeline is implemented in a single notebook.
